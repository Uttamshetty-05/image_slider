# 🖼️ Image Slider / Carousel

A modern and interactive **Image Slider (Carousel)** built using **HTML, CSS, and JavaScript**.
This project demonstrates smooth animations, user interaction, and dynamic UI updates.

---

## 🚀 Features

* ⏭️ Next / Previous navigation buttons
* ⚫ Dot indicators for slide position
* 🔄 Auto-slide functionality (every 3 seconds)
* 🎨 Modern UI with gradient background
* ✨ Smooth transition animations
* 🖱️ Clickable dots to jump to specific slides
* 💻 Fully browser-based (no setup required)

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ▶️ How to Run

1. Clone or download this repository
2. Open the `index.html` file in any browser
3. Use:

   * ⏭️ Next / Prev buttons
   * ⚫ Dots for navigation
4. Enjoy the smooth sliding experience 🎉
